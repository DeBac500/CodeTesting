Installation

  1. download the zip package from sourceforge
  2. unzip everything
  3. you are ready to start

Quick start

  1. launch startgui.bat
  2. click on the 'quotedb' button a quote list appears containing one GE entry.
  3. select the entry and click on ok.
  4. click on the 'load traders' button and open 'conf/traders.xml'
  5. click on the 'start' button robotrader is running...

Configuration files

Quotes file (conf/quotes.xml)

 <quotes basepath="conf/">
 	<instrument code="GE" startdate="19700202" enddate="20021018" filename="GE.txt" /> 
 </quotes>

Traders file (conf/traders.xml)

<?xml version="1.0" encoding="UTF-8" ?> 
<traders>
	<trader class="robotrader.trader.indicator.MFITrader">
  		<property key="PERIOD" value="15" /> 
  		<property key="BUYLVL" value="20" /> 
  		<property key="SELLLVL" value="80" /> 
  	</trader>
</traders>

Tickers file (conf/tickers.txt)

one line for each instrument:
MSFT
GE
