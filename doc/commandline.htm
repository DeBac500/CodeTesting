Command-line tool arguments

Launch the following on the command line: 
java robotrader.ConsoleMain xmlmasterquotefile xmltraderfile instrument|-all

Arguments:
  1. xmlmasterquotefile: xml file containing the path to the quotes for each instrument and the period available for the quotes
  2. xmltraderfile: xml file containing the list of Trader objects to use for the computation with their parameters
  3. instrument|-all: the instrument abbreviation as stores in the quotes data base, use -all to launch a computation for all instruments available in the quotes database and cumulate results of all computations
